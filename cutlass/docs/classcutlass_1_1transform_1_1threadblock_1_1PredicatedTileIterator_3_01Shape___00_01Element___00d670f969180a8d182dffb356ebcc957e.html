Detailed Description

template<typename Shape_, typename Element_, int AdvanceRank, typename ThreadMap_, int AccessSize, int InterleavedK>
class cutlass::transform::threadblock::PredicatedTileIterator< Shape_, Element_, layout::RowMajorInterleaved< InterleavedK >, AdvanceRank, ThreadMap_, AccessSize >

Specialization of PredicatedTileIterator for interleaved-32 data. It is mapped to the congruous layout.

Satisfies: ForwardTileIteratorConcept | ReadableContiguousTileIteratorConcept | WriteableContiguousTileIteratorConcept | MaskedTileIteratorConcept
